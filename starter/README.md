# Starter map

This directory is the smallest editable India Map Studio application. From the
repository root, run:

```text
python -m http.server 8000
```

Then open `http://localhost:8000/starter/`.

- Edit `index.html` to change the page structure.
- Edit `styles.css` to change the map surface and selected color.
- Edit `app.js` to react to map events or connect your data.

The starter references the repository's shared engine and national SVG. Keep
those relative paths, or copy the dependencies beside your application and
update the URLs.
