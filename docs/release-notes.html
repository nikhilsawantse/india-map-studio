First stable release

India Map Studio 1.0.0

A stable framework-free runtime, 36 state and union-territory maps, 750 district features, strict boundary metadata, and practical publishing examples.

Highlights

Stable APIIndiaMapEngine and <india-svg-map> Version 1 contracts. Public layers36 state/UT SVGs and 750 interactive district features. 23 recipesTwenty-two gallery demos and the multiple-map example, all with copy-ready starters. Quality gatesBrowser, accessibility, mobile, performance, schema, licensing, and release-package checks.

Release assets

The tag workflow publishes a complete source archive, a minimal starter archive, and SHA-256 checksums. Packages contain Git-tracked public assets only.

Data notice

Application code is MIT licensed. Geographic assets retain their source terms. Held local prototypes without confirmed redistribution rights are excluded, and published boundaries are for visualization rather than legal determination.

Upgrade and citation

Read the migration guide for pre-1.0 integrations. Research and published projects can use the repository's citation metadata.

Deferred

Localization infrastructure, offline support, and deeper nationwide local-boundary datasets remain outside Version 1.0.
